> 鸿蒙SDK是由ClkLog自主研发的鸿蒙APP用户行为埋点日志采集SDK。

### 2025-03-14 SDK初版发布

- 鸿蒙SDK相关功能
  - 标识用户
    - 自动标识匿名用户：自动使用设备ID标识匿名用户.
    - 标识登录用户：支持替换为自定义的匿名ID.
    - 设置用户属性.
  - 采集数据
    - SDK底层封装设备信息，屏幕高度、系统版本号等信息，在数据上报时自动采集此类数据并上报.
  - 数据上报
    - SDK 在满足一定条件后自动发送事件数据.
  - 调试功能
    - 开启调试日志，在控制台查看SDK采集的日志数据详情.
  - 自定义事件埋点
    - 在应用中对自定义事件及属性进行手动埋点.
